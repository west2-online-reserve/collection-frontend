定位模式

1.静态定位static

选择器{position:static;}

默认为静态定位，没有边偏移




2.相对定位relative

移动位置时，相对于自己原来的位置移动

移动后仍占有原来的位置

向右下移动分别100点
默认



3.绝对定位absolute

移动时，相对于定位父元素（祖先元素）移动

没有祖先元素时，以浏览器为准

父元素（或祖先元素没定位时，仍以浏览器为准）


          ⬇

“子绝父相”


定位冲突时，左 和 上 优先级更高

定位只改变设置定位的方向，未设置的和原来标准流的位置保持一致


所以，添加绝对定位时要给其所参考的元素添加（非静态）定位

绝对定位和浮动一样会脱标！！！

所以，使用 子绝父相，可以在不影响父元素标准流布局的情况下，使子元素自由地定位




4.固定定位fixed

可以固定于浏览器可视区的某一位置

脱标

固定于可视窗口的某一位置

脱标

特殊应用：将盒子固定在版心右侧

方法：定位在可视窗口左边的50%，再设置左边距为版心宽度的一半

版心



5.n粘性定位sticky

相对定位 和 固定定位 的混合
