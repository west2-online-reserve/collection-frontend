# JavaScript的定义
- JavaScript 是一种脚本编程语言，它可以在网页上实现复杂的功能，网页展现给你的不再是简单的静态信息，而是实时的内容更新——交互式的地图、2D/3D 动画、滚动播放的视频等等

# 怎样向页面添加 JavaScript
- 内部JavaScript：`<script>`在此编写 JavaScript 代码`</script>`
- 外部JavaScript：在html中添上`<script src="main.js"></script>`

# 注释
- 在双斜杠（//）后添加单行注释，比如：

  `// 我是一条注释`
- 在 /* 和 */ 之间添加多行注释，比如：

  `/* 我也是一条注释 */`
