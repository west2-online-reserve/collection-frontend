语句

1.顺序语句

2.分支语句

可以选择性地执行代码

1.if语句

if(条件1){

满足条件1要执行的语句

}else if(条件2){

满足条件2执行的语句

}else{

不满足上述条件执行的语句

}

除了0,空字符串,null,undefined都为真

2.三元运算符

条件?满足条件执行的代码:不满足条件执行的代码

一般用于取值

3.switch语句

switch(数据){

case 值1:

代码1

break

case 值2:

break

default:

代码n

}

3.循环语句

while循环

while(循环条件){

要重复执行的代码

}

for循环

for(整个循环前执行;循环条件;每次循环后执行){

循环体

}

for(变量起始值;循环条件;变量变化量){

循环体

}

循环的退出

while(true)和for(;;)都可以构造无限循环

break退出整个循环

continue结束本次循环,继续下次循环

遍历数组
