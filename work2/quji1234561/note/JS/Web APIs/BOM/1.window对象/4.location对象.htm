location对象

location的数据类型是对象,它拆分保存了url地址的各个组成部分

herf属性

获得完整url地址,赋值完成跳转操作

经常用location的herf属性,利用js跳转页面

还有60s跳转到百度

search属性

获得地址中的参数,符号?后面部分

场景:获得表单信息

如 '?userName=Leo&pwd=123456'

hash属性

获取地址中的哈希值,符号#后面部分

可实现页面不需要跳转也能切换内容

给a加herf="#/哈希值"

reload()方法

重新加载页面

location.reload(true)类似ctrl + F5,强制刷新
