本地存储介绍

为了在本地存储大量数据

1.将数据存储在用户浏览器中

2.设置、读取方便、甚至页面刷新不丢失

3.容量较大,sessionStorage和localStorage约5M左右

本地存储分类

localStorage

能够使用localStorage把数据存储到浏览器中

作用:永久存储在本地,除非手动删除,否则关闭界面也会存在

特性:可以多窗口(页面)共享(同一浏览器,且不能跨域);以键值对的形式存储使用

1.存储(改)

语法:

localStorage.setItem(key,value)

key要加引号,否则会被当作未定义的变量

本地存储只能存字符串,数字会转为字符存储

改 要重新赋值

2.获取

语法:

localStorage.getItem(key)

3.删除

语法:

localStorage.removeItem(key)

sessionStorage

与localStorage最大不同:关闭浏览器数据会消失

使用方法基本相同
