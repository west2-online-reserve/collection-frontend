# 悦读 / Joy Reader

仓库地址：https://github.com/PrinceSaoKe/Joy-Reader

## 亮点

- 完全使用TypeScript
- 用class结合interface、enum定义数据模型
- 简单封装了接口
- 使用正则表达式剔除文章中的markdown语法生成文章简介，剔除html标签如<script>防止XSS攻击
- 使用拦截器实现Token过期跳转登录页面
